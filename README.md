# Automatic language detection and translation program using the Google Translate API

## Overview
This program automatically detects the language of a user-entered sentence and translates it to another language specified by the user. The program uses the Google Translate API to perform the translation.

## Usage
1. The user is prompted to enter a sentence they wish to translate.
2. The program detects the language of the sentence.
3. The user is prompted to enter the 2- or 3-letter code for the language they want to translate the sentence to.
4. The program verifies that the user-entered code is valid and on the list of supported languages.
5. The program uses the Google Translate API to translate the sentence to the desired language.
6. The translated sentence is printed to the console.

## Dependencies
This program requires the `googletrans` package to be installed. It can be installed using pip:

pip install googletrans==4.0.0-rc1

## Code Explanation
1. The user is prompted to enter a sentence they wish to translate using `input()` function.
2. A dictionary `language_list` is created with keys as language codes and values as the full names of the languages.
3. The program prints out the supported languages and their codes.
4. The user is prompted to enter the language code of the language they wish to translate the sentence to.
5. The program verifies that the user-entered code is valid and on the list of supported languages. If not, it will prompt the user to enter a valid code.
6. The `Translator()` function from the `googletrans` package is used to create a translator object.
7. The `translate()` method of the `translator` object is used to translate the sentence to the desired language.
8. The translated sentence is printed to the console using `print()` function.
